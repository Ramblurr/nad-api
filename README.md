# nad-telnet-to-rest

Simple HTTP API to connect to NAD receivers through the RS232 interface, Telnet interface or the TCP/IP interface.
For the RS232 interface use the NADReceiver class For the Telnet interface use the NADReceiverTelnet class For the TCP/IP interface use the D7050 class

Note that the telnet interface is only tested with the NAD T778.
Commands are implemented based on the T778.
Those commands should work with more NAD receivers.
